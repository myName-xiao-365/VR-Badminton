# VR Badminton

这是一款像素风羽毛球游戏原型。

## 启动游戏

1. 解压完整构建文件夹。
2. 运行游戏可执行文件。
3. 在左侧选择单人模式、难度和分制。
4. 联机模式目前为功能占位。

Sensor 模式需要 Windows 摄像头、手机与电脑处在同一局域网，并允许游戏进程访问
摄像头和入站 TCP 端口。当前 Windows Player 会从
`VR Badminton_Data/StreamingAssets/VRBadminton/MediaPipe/` 读取
`pose_landmarker_lite.bytes`。

## 基础操作

| 按键 | 功能 |
| --- | --- |
| `W A S D` | 控制人物移动 |
| 鼠标上下移动 | 调整球拍拍面角度 |
| 鼠标纵向挥动 | 挥拍 |
| `Q` | 切换正手与反手 |
| `Space` | 接杀准备、高远球起跳或确认对手发球 |
| `Esc` | 暂停游戏 |

## 主界面

- 右侧 `ENTER MATCH` 进入比赛选择。
- 可选择继续已有比赛或新建比赛。
- 新建比赛时选择模式、难度和 15/21 分制。
- 比赛开始后不能再修改难度和分制。
- 返回主界面会保留比分与发球权。
- 正在进行的一球会回滚，继续时由原发球方重新发球。
- 新手教程目前为占位功能。

黄色圆圈表示羽毛球落点，青色方块表示建议人物站位。
地面彩色虚线表示从击球点到预计落点的轨迹投影。
地面青色短横线表示当前球拍拍面中心位置。
可在 `Settings` 中关闭或开启拍面辅助线。所有球路都会显示实时地面投影。

## 设置

右上角 `Settings` 可以选择：

- 自动适应当前显示器分辨率的全屏模式
- 窗口模式
- 1280 × 720
- 1600 × 900
- 1920 × 1080
- 2560 × 1440（2K）

详细玩法请阅读《游戏操作指南与规则》。
